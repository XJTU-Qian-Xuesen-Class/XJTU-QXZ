#  跳绳

## 简介
1. 上课学期：2024秋+2025春。
2. 主要内容：主要分为竞速跳绳和花样跳绳两部分。
3. 学分：0.5。

## 授课内容
每节课都有竞速跳绳的训练部分（1分钟双脚交换跳的定速训练）、花样跳绳的教学部分（秋季学期学习花样跳绳一段套路动作，春季学期学习花样跳绳二段套路动作）和拓展部分（更高难度动作学习、专门针对春季跳绳比赛的训练等，但均非期末考核内容），其中高丹娜老师的课上关于更高难度动作的内容更多一些。

## 考核方式
专项考核均为20分竞速跳绳和20分花样跳绳。竞速跳绳满分标准为30秒100下（秋季学期）/1分钟200下（春季学期），未满分则按次数等比例给分；花样跳绳黄霞老师给分比高丹娜老师普遍高1-2分，正常练习都可以拿到16-19分。

## 注意事项
跳绳课每年第一轮选课的人数都不多，比较好抢，对于相对没有体育细胞的同学来说，跳绳课算是一个比较好的选择，给分也不错，欢迎选课。
