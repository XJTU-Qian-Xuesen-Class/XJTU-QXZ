# 体适能

## 简介
1. 上课学期：2024秋+2025春。
2. 主要内容：提升学生体能，锻炼基本素质。
3. 学分：0.5。

## 授课内容
体适能每节课的授课内容比较固定，无非就是对于肌肉力量的练习，弹跳力量有关动作的培训，以及对于跑动中的一些基本动作的练习。因此这在授课上算是一门相对比较无聊的课程，但是正在这种比较无聊但是又相对比较有压力的训练下，我们的身体素质才得以比较有效的加强。虽然说这种训练比较有压力，但是是根据每个同学自身的身体素质来的，不会说难到要让身体素质比较差完成不了。

## 考核内容
秋季学习的考核内容是1500米和平板支撑，春季学期的考核内容是4×10米折返跑和后抛实心球。一般来说，秋季学期的得分会比春季学期要高很多。别被1500米吓住了，他的满分标准真的不高，只要你的1千米或800米体测成绩能够达到70分左右，这个就能拿到满分了。但是春季学期的两个标准，上过的同学认为还是很高的。

## 注意事项
相比于需要技巧的球类，体适能这门课程可谓是简单又强势。一般来说只要是一个正常体质的大学生，就能在这个课程中拿到一个不错的分数，而且提升自己的身体素质。但是在竞争激烈的羽毛球和乒乓球中，可能打的一般的同学就拿到一个比较不好看的分数了。
