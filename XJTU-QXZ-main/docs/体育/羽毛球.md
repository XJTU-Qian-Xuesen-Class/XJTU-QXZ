# 羽毛球

## 简介
1. 上课学期：2024秋+2025春。
2. 课程简述：羽毛球的基本动作（2024年秋）和实战技巧（2025年春）。
3. 学分：0.5。

## 授课内容
体育一羽毛球主要学的是发球、正手高远球等基本动作，老师讲解得比较详细，0基础也可以学。体育二羽毛球主要学的是吊球、杀球等实战技巧。

## 考核方式
体育一羽毛球的考核方式是20分对打高远球（打10个就能15分合格分以上，姿势标准、球速快加分），20分四角移动（按时间多少给分）。体育二羽毛球的考核方式是24人分4组组内打比赛，第一名满分，之后每低一名扣2-3分。

## 注意事项
对于有羽毛球基础的同学来说无论是体育一还是体育二的羽毛球都非常轻松，但没基础的同学也可以尝试体育一羽毛球，老师教的比较循序渐进，体育一的考核也不算太难，认真练习可以拿到高分。体育二的成绩有点依赖分组运气和个人发挥，建议对自己水平有信心/愿意付出时间练习羽毛球技术/不太在意体育分数的同学选择。
