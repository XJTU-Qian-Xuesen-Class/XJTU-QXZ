# 艺术欣赏与创作陶艺
## 课程简介
- 上课学期：2024秋
- 课程简述：陶艺作品的创作实践
- 学分：**1分**

## 授课内容
- 理论教学：共4课时（约半天），老师简要介绍陶艺的历史与文化价值
- 实践教学：共28课时，在田家炳陶艺教室进行
- 实践要求：需完成至少1件陶艺作品制作+1个盘子图案烤制
- 材料工具：老师提供陶泥、颜料等材料及相关工具
- 作品处理：作品烤制完成后，老师会拍照打分，同学可将作品带回

## 考核方式
- 考核依据：以完成的陶艺作品作为作业
- 分数情况：给分较高，基本在85分以上

## 注意事项
- 问题咨询：老师经验丰富、态度温和，制作中遇问题先咨询老师，避免盲目操作导致作品瑕疵
- 加分机会：完成至少1件作品后仍有剩余时间，可尝试机器拉坯；若成功用该方式做出作品，成绩会大幅提高
- 选课建议：课程需占用两个周末，不介意此时间安排的话推荐选择
