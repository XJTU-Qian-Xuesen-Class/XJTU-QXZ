# 第三章 艺术欣赏与创作音乐
## 课程简介
- 上课学期：2024秋
- 课程简述：涵盖乐理知识、作曲技巧、声乐技巧相关教学
- 学分：**1分**

## 授课内容
- 基础教学：老师先讲解基本乐理知识
- 创作实践：在乐理基础上介绍作曲技巧，组织同学们分组创作音乐
- 声乐训练：教授声乐技巧，根据个人嗓音特质分声部练习合唱曲目
- 结课展示：举办结课音乐会，进行大合唱表演
- 个人展示：有乐器特长的同学可报名音乐会个人节目，经老师审核后可登台表演

## 考核方式
- 成绩构成：创编作业（15%）+ 结课表演（85%）
- 基础分数：结课表演均能获得80分以上基础分
- 额外加分：参与个人展示节目或音乐会筹办工作可获得额外加分

## 注意事项
- 基础适配：有音乐基础的同学学习难度较低，有较多展现自我的机会；无音乐基础的同学难度稍高，但能学到实用知识
- 参与途径：无乐器特长的同学可参与音乐会工作，可报名宣传组或主持人岗位
- 选课建议：推荐对音乐感兴趣的同学选择
