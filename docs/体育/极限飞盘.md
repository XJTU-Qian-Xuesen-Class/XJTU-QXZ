# 极限飞盘

## 简介
1. 上课学期：2025春。
2. 主要内容：极限飞盘的基本动作。
3. 学分：0.5。

## 授课内容
极限飞盘运动的基本动作，包括正手直线盘，反手直线盘等。

## 考核方式
考试为两人一组进行，在场地内互相传盘，需要完成正反手直线传盘各7次，65秒满分，65-80秒内每3秒扣一分，超过80秒每5秒扣一分。

## 注意事项
本课程需要自备飞盘。考试满分有较大难度，需要一定的练习与配合，建议尽早找到自己的考试搭子方便课上的时候直接练习。此外，飞盘课需要参加至少6次的飞盘练习打卡，时间为周一、二、四（2024-2025学年第二学期是这么安排的）晚上东南足球场，每次练习完后会有打卡二维码。不参加或参加次数不够对成绩会有影响。
