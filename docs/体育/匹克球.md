# 匹克球

## 简介
1. 上课学期：2024秋+2025春。
2. 主要内容：匹克球是一种新兴的隔网对抗球类运动，是多种球类运动的“结合体”。简单点说，可以理解为：在羽毛球场用大号的乒乓球拍打网球。我们学校在2024年秋才开设了匹克球课程（至今只上了两个学期），因此课程内容和考核方式还在探索中，因此本选课指南仅供参考，具体情况以本学期授课老师的通知为准。
3. 学分：0.5。

## 授课内容
匹克球-I的授课内容主要有：
1. 球性练习（颠球和一些辅助训练等，两节课左右）
2. 发球
3. 正反手击球
4. 丁克球（在己方非截击区（球网后的一片区域）后，用极轻柔的力量把球吊到对方非截击区）

匹克球-II的授课内容主要有：
1. 底线击球
2. 网前截击（球不落地直接击球称作截击）
3. 简单战术（接发球上网等）
4. 挑高球和高压球等技术

## 考核方式
匹克球-I的考核为：
和搭档打丁克球，打十个回合。分数构成为：20分回合分（比如打满了十个回合就可以得满）和20分技术分（老师通过观察你的脚步运用、正反手技术动作来给分）。

匹克球-II的考核为：
老师根据水平将班里同学分成若干组，每组4人，然后组内打单循环比赛，根据排名和技术动作给分。第一名给36~40，第二名给32~35，第三名给28~31，第四名给24~27。

## 注意事项
总的来说一句话：大胆去报就可以

首先，匹克球学习成本很低，上手很快，对于有羽毛球、乒乓球或网球基础的同学来说更为容易，因此不用担心学不会、打不好。

然后，匹克球-I的给分很高，老师人特别好，会尽可能帮助你，让你和搭档打满十个回合（正式考试每人有两次机会），而其中20分的技术分，老师也是能给高就给高，大家最后分差不大，都是36~38左右。

另外，匹克球课的运动强度不大。这一方面是匹克球这项运动本身的特点（球速慢、跑动少）决定的，另一方面是因为上课基本是四个人共用一块场地，跑动的范围不大，可能整节课最累的就是课前的热身跑。

最后，匹克球抽中的概率高，课好抢。由于开课时间不长，很多同学甚至都不知道有这门体育课的存在，即使看到了，也会因为不了解这项运动，不敢去尝试新鲜事物而放弃这门课。大部分的匹克球课，都可以在一轮选课的时候直接选中，到二轮选课依然有很多的空位。因此，欢迎同学们将匹克球列入自己体育课的备选清单，期待与同学们在球场上见面~
