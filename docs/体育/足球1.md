# 足球I

## 简介
1. 上课学期：2024秋。
2. 主要内容：足球的基本动作。
3. 学分：0.5。

## 授课内容
主要学习脚背内侧踢球，传球，射门等基本技术动作，老师讲解比较详细，零基础可学，但可能需要自己在课下继续练习。

## 考核方式
在一定距离（体育I为11米，体育II为13米）上面对空门踢点球，共计5个球，根据踢进不同的个数给分（满分30分），同时根据踢球的技术动作给出技评分（满分10分），要求球在过门线时必须离地。

## 注意事项
对于有一定足球基础的同学比较轻松，无此类基础的同学要想考试通过可能需要课下练习，老师讲解很详细，有不懂的地方一定要问老师。此外，每节课最后大约会留20-30分钟分组模拟比赛。
