# 走进天文学
## 课程简介
- 上课学期：2025春
- 主要内容：天文学相关知识讲解
- 学分：**2分**

## 授课内容
核心涵盖天文学基础内容，包括宇宙发展史、恒星、银河系等相关简介。

##  考核方式
- 成绩构成：考勤与阶段性随堂作业（50%）+ 课程论文（50%）
- 考勤与随堂作业：课程中不定期布置，共5-6次左右；需写在纸上提交，内容与课程相关（如人类对宇宙认知的七次飞跃、对宇宙大爆炸的理解等）
- 课程论文：
  - 选题方式：老师选定10个左右主题，课堂上简要讲解各选题思路；通过助教发布的共享文档填写选定主题，每个主题名额有限
  - 提交要求：完成后提交论文即可

## 注意事项
- 评分核心：成绩主要依据论文评定
- 成绩范围：论文质量优秀者基本能获得90分以上；论文表现一般但考勤、课堂作业均参与的同学，成绩不低于80分
