# 新能源导论
## 课程简介
- 上课学期：2024秋
- 主要内容：包含环境问题及能源利用效率、新能源利用技术、储能技术三部分
- 开课单位：电气工程学院
- 学分：**2分**

## 授课内容
- 吴锴老师：讲解世界环境问题、环境经济学基本原理等
- 吴坚老师：讲解风能、核能、太阳能等各类新能源的发电相关知识
- 周峻老师：讲解储能技术在新能源中的应用，课程中会安排一次小测
- 作业安排：吴锴老师、周峻老师各布置一次作业，每次含三道简答/计算题

## 考核方式
- 考核要求：提交1份上机报告（GAMS编程计算能源系统优化问题）+ 1篇论文

## 注意事项
- 作业评分：吴锴老师的作业给分方式较抽象，但不影响最终得分
- 评分核心：成绩主要依据上机报告和论文，打分中规中矩
- 考勤说明：课程不点名
