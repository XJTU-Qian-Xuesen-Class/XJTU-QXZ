# 全球气候史
## 课程简介
- 上课学期：2024秋
- 主要内容：全球气候相关知识讲解
- 学分：**2分**

## 授课内容
核心涵盖全球气候的发展历程、突变原因，以及气候变化所带来的影响等。

## 考核方式
- 成绩构成：课堂表现（30%）+ 专题调研综述论文（70%）
- 课堂表现：每节课课间通过微信签到+手机作答测试题，需提交包含PPT的教室内自拍；测试题多为1道单选或多选，难度较低
- 专题调研综述论文：
  - 主题要求：需与本课程内容相关
  - 参考文献：在中国知网、Scopus等网站查阅不少于10个
  - 内容要求：整理文献观点后给出个人评述，或利用文献资料证明个人观点，字数不少于3000字
  - 格式要求：参考《西安交通大学学报》格式

## 注意事项
- 评分核心：成绩主要依据论文评定
- 成绩范围：论文质量优秀者基本能获得90分以上；论文表现一般但考勤、课堂作业均参与的同学，成绩不低于80分
