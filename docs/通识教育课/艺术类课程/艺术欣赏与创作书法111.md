# 艺术欣赏与创作书法
## 课程简介
- 上课学期：2024秋
- 课程简述：涵盖楷书、行书等毛笔字练习，及书法欣赏相关内容
- 学分：**1分**

## 授课内容
上课以学生自主练习书法为主，课程开头与中间阶段，老师会穿插讲解书法知识及写字技巧。

## 考核方式
- 课程周期：16周，每周1次课
- 成绩构成：平时成绩（30分）+ 期末成绩（70分）
- 平时成绩：课程期间随机2次课，上交1张课上书写的毛笔字即可获得满分
- 期末成绩：最后1次课现场创作1幅作品，由老师评价打分

## 注意事项
- 自备物品：毛笔、宣纸、砚台、毛毡
- 考试用纸：期末考试作品建议使用书法比赛专用作品纸（老师期末前会提醒），无需使用平时练习宣纸
- 成绩说明：课上认真练习即可，无书法基础也基本能取得80%以上成绩，作品仅需书写两句话
